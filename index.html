Dynamic Array Tools

Dynamic Arrays are special arrays in Maya that you can use with the instancer node. This means you can generate points and the instancer will display meshes instead of points. This toolset will help you manipulate the points in a non-destructive matter.

Note: This is a toolset that is still under development
